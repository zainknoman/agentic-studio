## Review cadence

- **Weekly check-in** — update "Current progress" for each KR; flag blockers
- **Mid-quarter review** — re-forecast targets if assumptions have changed; do not lower targets
  without a documented reason
- **End-of-quarter retrospective** — score each KR (0–100%), capture learnings, set next OKR cycle

> A KR score of 60–70% is considered a success. 100% means the target was set too conservatively.
> 0–40% requires a root cause analysis before the next cycle.
